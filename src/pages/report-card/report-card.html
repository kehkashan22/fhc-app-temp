What is A, B, C breakup?

On the basis of exam papers of past 10 years, Farooq Sir has devised a scientifically accurate analysis system. In this system all chapters of each subject are divided into three categories: A, B and C.

On an average (as seen in the past 10 years), the breakup of scores for each topic is as follows:

Chapter Type Marks (out of 100)
A 60
B 30
C 10

Similarly, each chapter has further been divided into three sub-categories:

Application, Speed and Memory,

with different weightage alloted to each category in each chapter type.

  1. In category A topics most questions are based on Application and only a few on Memory and Speed

  2. In category B topics most questions are based on Memory and only a few on Application and Speed

  3. In category C almost all topics are Memory based

This test series has been devised to accurately project the aggregate marks a student should expect in their CA/CS exam, based on the number of quizzes of each type and sub-type solved. Plus, for each quiz, chapter and type, individual average scores have also been calculated.

So one must try to give as many quizzes as possible for an accurate assessment.
