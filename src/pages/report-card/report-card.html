What is A, B, C breakup?

On the basis of exam papers of past 10 years, Farooq Sir has designed an analysis system keeping in mind the statistics that came out from past experience and keen understanding of the working of the exam. In this system all chapters of each subject are divided into three categories: A, B and C.

On an average (as seen in the past 10 years), the breakup of scores for each topic is as follows:

Chapter Type Marks (out of 100)
A 60
B 30
C 10

Similarly, each type has further been divided into three sub-types:

Application, Speed and Memory,

with different weightage alloted to each category in each chapter type.

  1. In topics of type A most questions are based on Application and only a few on Memory and Speed.

  2. In topics of type B most questions are based on Memory and only a few on Application and Speed.

  3. In topics of type C almost all topics are Memory based.

This test series has been devised to accurately project the aggregate marks a student should expect in their CA/CS exam, based on the number of quizzes of each type and sub-type solved. Plus, for each quiz, chapter and type, individual average scores have also been calculated.

Solve as many quizzes as possible for the best assessment. Best of Luck and happy Learning!
